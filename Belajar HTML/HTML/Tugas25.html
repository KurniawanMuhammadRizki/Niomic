Mengenal Sematic Elements Di HTML 5

  • Article

    Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita gunakan dengan element header dan footer. karena tag article memiliki tipe block. artinya element ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bawah postingan karena tag aside memiliki tipe block. artinya element ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.
